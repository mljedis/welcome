## Python, Keras

### macOS

- install homebrew https://brew.sh/
  - `brew` should work
- install python3 (really needed?)
  - `brew install python3`
  - `brew link python3`
  - `python3` should work
- install pip
  - `easy_install pip`
  - `pip` should work
- install keras
  - `pip install keras --user`
